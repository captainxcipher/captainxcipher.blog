Cryptography: Ciphered Text in C
CS50 Project 2
By Captain X Cipher  |  July 2025

Introduction

Cryptography means converting plaintext to code words (cipher) using a key. In this CS50 project, youâ€™ll learn how to write a C program that converts plain text to ciphered text using a 26-letter alphabetical key.

Step 1: Command Line Argument for Key

We use a command line argument for the key. This code ensures the user provides a valid 26-character key containing only letters, with no repeats. 


int main(int argc, string argv[])
{
    if (argc != 2 || strlen(argv[1]) != 26)
    {
        printf("Usage: ./substitution key or Key must contain 26 characters\n");
        return 1;
    }
    int len = strlen(argv[1]);
    for (int i =0;i < len; i++)
    {
        if (!isalpha(argv[1][i]))
        {
            printf("Key must contain only letters\n");
            return 1;
        }
        for (int j = i + 1; j < len; j++)
        {
            if (toupper(argv[1][i])== toupper(argv[1][j]))
            {
                printf("Letters cannot be repeat\n");
                return 1;
            }
        }
    }
}

Step 2: Enciphering Plain Text

This function takes the key and the text, and converts the plain text to ciphered text. Each letter is substituted according to the key. 


string enciphertext(string text,string key)
{
    int len = strlen(text);
    string ciphertext = malloc(len+1);
    int index;
    for (int i = 0; i < len; i++)
    {
        if (isupper(text[i]))
        {
            index = text[i] - 65;
            ciphertext[i] = toupper(key[index]);
        }
        else if (islower(text[i]))
        {
            index = text[i] - 97;
            ciphertext[i] = tolower(key[index]);
        }
        else
        {
            ciphertext[i] = text[i];
        }
    }
    ciphertext[len] = '\0';
    return ciphertext;
}
