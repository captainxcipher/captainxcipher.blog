Credit Card Checker: C Language
CS50 Project 1
By Captain X Cipher  |  July 2025

Step 1: Prompting User for Input

First, we prompt the user to enter a positive credit card number using get_long from the CS50 library. 


long number;
do
{
    number = get_long("Number: ");
}
while (number < 1);

Step 2: Counting the Length of the Number

We count the number of digits to help identify the card type.
American Express uses 15 digits, MasterCard uses 16, and Visa uses 13 or 16. 


int length_count(long number)
{
    int length = 0;
    while (number > 0)
    {
        number /= 10;
        length ++;
    }
    return length;
}

Step 3: Validating with Luhn’s Algorithm

  • Multiply every other digit by 2, starting with the second-to-last digit, and add those products’ digits together.
  • Add the sum to the sum of the digits that weren’t multiplied by 2.
  • If the total’s last digit is 0 (i.e., total modulo 10 is 0), the number is valid.

char check_valid(long number)
{
    int sum1 = 0;  // digits not multiplied by 2
    int sum2 = 0;  // digits multiplied by 2
    int multiply = 0;
    while (number > 0)
    {
        sum1 += number%10;
        number /= 10;
        multiply = number%10;
        multiply *= 2;
        while (multiply > 0)
        {
            sum2 += multiply%10;
            multiply /= 10;
        }
        number /= 10;
    }
    int valid = sum1 + sum2;
    if (valid%10 == 0)
    {
        return 'Y';
    }
    else
    {
        return 'N';
    }
}

Step 4: Identifying Card Type

After checking validity, we identify the card type by its length and starting digits:

  • American Express: 15 digits, starts with 34 or 37
  • MasterCard: 16 digits, starts with 51–55
  • Visa: 13 or 16 digits, starts with 4

string card_type(long number, int length)
{
    int n1 ;
    if (length > 12 && length < 17 && length != 14)
    {
        if (length == 16)
        {
            n1 = number/100000000000000;
            if (n1 > 50 && n1 < 56)
            {
                return "MASTERCARD";
            }
            else if (n1 > 39 && n1 < 50)
            {
                return "VISA";
            }
            else
            {
                return "INVALID";
            }
        }
        else if (length == 15)
        {
            n1 = number/10000000000000;
            if (n1 == 34 || n1 == 37)
            {
                return "AMEX";
            }
            else
            {
                return "INVALID";
            }
        }
        else if (length == 13)
        {
            n1 = number/1000000000000;
            if (n1 == 4)
            {
                return "VISA";
            }
            else
            {
                return "INVALID";
            }
        }
        else
        {
            return "INVALID";
        }
    }
    else
    {
        return "INVALID";
    }
}
